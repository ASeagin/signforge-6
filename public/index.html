How everything works

Where user data is stored, how payments reach you, and what each account does.

👤

User Accounts & Data Storage

Right now (development mode)

Accounts are stored in server memory — this means they work perfectly while the server is running, but they reset when you restart the server. This is normal for development and testing. Your first 100 test users will be fine.

When you go live (production)

Accounts will be stored in a PostgreSQL database (like a permanent spreadsheet that never resets). The easiest free option is Supabase.com — free up to 50,000 users. I can upgrade the code for you when you're ready.

What gets stored
✓ Name & email
✓ Hashed password
✓ Plan (free/starter/pro)
✓ Trial used or not
✓ Document history
✓ Saved signature
What is NOT stored
✗ Actual password (only hash)
✗ Card details (Stripe handles)
✗ PDF documents (deleted in 24h)
✗ Any tracking data
✗ IP addresses
Password security
Passwords are hashed with bcrypt before storage. This means even if someone stole the database, they could never see real passwords. It's industry standard security used by all major apps.

On $1.99 → you keep $1.63
On $4.99 → you keep $4.55
When you get paid
Stripe pays out to your bank account every 2 business days automatically. You can also do instant payouts for a small fee.
You never touch cards
Stripe handles all card security. You never see card numbers. This is why major companies use Stripe — it's the safest option.
📋 Step-by-step: Set up Stripe to receive money
1
Go to stripe.com and create a free account with your email
2
In the Stripe dashboard go to Settings → Bank accounts → add your bank account (sort code + account number)
3
Go to Developers → API Keys → copy your Secret Key and Publishable Key
4
Open your .env file in VS Code and paste in both keys where it says REPLACE_ME
5
Go to Products in Stripe → create "Starter $1.99/month" and "Pro $4.99/month" → copy the Price IDs into .env
6
Restart the server (Ctrl+C then npm start) — payments now work and money goes to your bank! 🎉
